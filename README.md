# 🚀 AI-Powered Placement Preparation Platform

An intelligent placement preparation platform designed to help students improve their interview skills through AI-powered mock interviews, coding practice, aptitude tests, and personalized performance analysis.

---

## 📖 Overview

The AI-Powered Placement Preparation Platform provides a complete environment for students preparing for campus placements. It combines aptitude practice, coding challenges, technical interviews, HR interviews, and group discussions into a single platform.

The system evaluates user performance, provides instant feedback, and recommends areas for improvement using Artificial Intelligence.

# ⚙ Installation

## Clone the Repository

```bash
git clone https://github.com/Nivetha112006/AI-powered-placement-preparation-platform.git
```

Navigate to the project folder

```bash
cd AI-powered-placement-preparation-platform
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Run backend server (if applicable)

```bash
npm run server
```
